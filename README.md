# SEO pipline
##The project is designed to retrieve data from the Yandex Wordstat API to determine the frequency of words in user queries in Yandex search.
### (Проект предназначен для получения данных с API (Яндекс Вордстат) для определения частоты слов в пользовательских запросах в Яндекс-поиске.)
![alt text](images/miro.jpg)

### Start project:

Clone the repository and switch to it on the command line:

```
bash
https://github.com/taren4ik/wordstat
cd wordstat
```

Create and activate a virtual environment:

```
bash
python -m venv venv
```

For *nix systems:
```
bash
source venv/bin/activate
```

For windows systems:
```
bash
source venv/Scripts/activate
```
Install dependencies from the requirements.txt file:

```
bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```
Create a .env file and specify environment variables:

Start project:

```
bash
python main.py
```
